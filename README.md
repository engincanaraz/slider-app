# Araç Slider Projesi

Bu proje, araçları gösteren interaktif bir slider uygulamasıdır.

## Özellikler

- Otomatik slider geçişi
- Manuel kontrol için ok butonları
- Hover efektleri
- Responsive tasarım
- Smooth geçiş animasyonları

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- Bootstrap 4
- Font Awesome
